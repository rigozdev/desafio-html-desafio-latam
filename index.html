Sobre mí

Soy analista programador, titulado el 2021. Actualmente me encuentro realizando una serie de cursos para mejora mi entendimiento y habilidades como desarrollador web o frontend. Dentro de ellos me encuentro realizando un curso de javascript fullstack en 'Desafío Latam'. Cuento con certificados de fundamentos en Javascript, pero considero que aun tengo mucho por aprender. Me gusta mucho trabajar en equipo porque considero que de esa forma se puede tener mucho mejor retroalimentación y ayuda de colegas, intercambiando concocimientos y aplicandolos para en la práctica ir mejorando día a día. Tengo conocimientos de programación orientada a objetos con lenguajes como Java y C#. Conocimiento de bases de datos relacionales SQL. Mi principal motivación es poder crear y aportar en proyectos que sean de gran ayuda para resolver problemas y también resolver problemas de soluciones ya realizadas. Cnosidero que gracias al desarrollo y diseño web es posible realizar tareas de una manera mucho más rápida para el usuario común, entender y persuadir al usuario para que pueda entenderse mejor con aplicaciones es una muy buena forma de realizar un aporte resolviendo de forma más rápida los distintos problemas o requerimientos. Por úttimo dentro de mis principales metas está el poder dominar React con distintos frameworks, pero mis objetivos están orientados a Remix y Next, y estos conocimientos combinarlos con una buena base para una correcta maquetación
